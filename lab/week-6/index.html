For this website, I used the display:grid function to create an invisible grid that seperates the three main content elements:number, text, and photo.
I used the display:flex function as a style element in my container for the grid, in order to align the grid to the center of the webpage. The position:fixed function was used in the title, where it stays on the top as is always visible even while scrolling down. I was struggling to use position:relative in the designing, so I added it here in my explanation!
